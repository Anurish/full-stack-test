# WPoets Full Stack Test

A PHP + MySQL CRUD solution with a responsive frontend section that matches the provided design requirements.

## Features

- CRUD for tabs and slides
- MySQL schema with sample seed data
- Desktop layout with:
  - tabs in column 1
  - synced content slider in column 2
  - synced image slider in column 3
- Mobile layout with accordion tabs and background-image slider cards
- jQuery interactions and Bootstrap styling
- Clean typography using Open Sans and Titillium Web

## Setup

1. Create a database named `wpoets_full_stack_test`.
2. Import `database.sql`.
3. Update credentials in `config/config.php`.
4. Place the project in your web root.
5. Open `index.php` for the frontend.
6. Open `/admin/index.php` for CRUD management.

## Notes

- Sample SVG icons and images are included in `assets/images`.
- The project uses local image files by default, so it works without external media.
